3. CONFIGURE DEPLOYMENT:

   • docker compose build
   • docker compose up -d

5. INITIALIZE:
   • Run database initialization script
   • Create MinIO bucket
   • Create admin user

6. VERIFY:
   • ./deployment-scripts/health-check.sh
   • Access https://yourdomain.com
   • Test API at https://api.yourdomain.com
