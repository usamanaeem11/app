# Chrome Extension Icon Assets Required

Create the following PNG icon files in this directory:

## Required Files:

- **icon16.png** - 16x16 pixels (toolbar icon)
- **icon32.png** - 32x32 pixels (extension management page)
- **icon48.png** - 48x48 pixels (extensions page)
- **icon128.png** - 128x128 pixels (Chrome Web Store, installation)

## Design Guidelines:

- Format: PNG with transparency
- Background: Green (#10b981) or transparent
- Icon: White stopwatch/clock symbol
- Style: Simple, clear, recognizable at all sizes
- Brand consistent with WorkMonitor

## How to Create:

1. Design base 128x128 icon
2. Use https://resizeimage.net/ to create all sizes
3. Or use Figma/Photoshop to export all sizes
4. Save all 4 files in this directory

## Testing:

After adding icons:
1. Reload extension in chrome://extensions
2. Icons should appear in toolbar and management page
3. Test on light and dark backgrounds
