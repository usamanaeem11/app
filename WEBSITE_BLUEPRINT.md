### üéØ Global Linking Rules (Non-Negotiable)

1. **Descriptive anchor text** - "employee monitoring software" not "click here"
2. **Link upward more than sideways** - Feed authority to core pages
3. **Max 3 clicks from Home** - Every page accessible quickly
4. **Every new page links to 2+ core pages** - No dead ends
5. **Tools always link to features** - Convert free traffic

### üîù Header & Footer = Authority Distributors

**Header Links (Strong Signal):**
- Only top-level feature pages
- Consistent across all pages
- Pass strong SEO authority

**Footer Links (Quieter but Scale):**
- All core features
- All industries
- Key tools
- Comparisons
- Ensures zero orphan pages
